This is my first post. The blog is built with [Jekyll](https://jekyllrb.com)
and hosted for free on [GitHub Pages](https://pages.github.com).

## Writing a new post

Add a Markdown file to the `_posts/` folder, named with a date prefix:

    _posts/YYYY-MM-DD-some-title.md

Begin the file with front matter (the block between the `---` lines), then write
your content in Markdown below it:

    ---
    layout: post
    title: "My post title"
    date: 2026-06-28 12:00:00 +0300
    categories: tag1 tag2
    ---

    Your content here. Markdown works — **bold**, _italic_, links, lists,
    and fenced code blocks.

Commit and push to the `main` branch — GitHub Pages rebuilds the site
automatically within a minute or two. That's the whole workflow.

Happy writing.
